To run this project Unity 5 is required.
Open DemoGame.unity to start the project
DemoGame.unity is located in TeamAI-master\TeamAI\Assets\DemoGame.unity
It is best to save the code on your local hard drive before running or else Unity will try and write certain files to the CD, which causes errors.

When the project is loaded press the play button in Unity to start a new match.
Press [S] to toggle showing of the strategy grid
Press [I] to toggle showing of influence grid
Press [P] to toggle showing of play lines
